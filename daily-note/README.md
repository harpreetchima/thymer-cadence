# Daily Notes

This package is the working copy of the Cadence-upgraded Thymer `Daily Notes` journal plugin.

Role:

- preserve Thymer's native daily journal behavior
- extend the native top navigation row with compact links to the matching week, month, quarter, and year records
- upgrade the native inline date picker so it can jump across daily, weekly, monthly, quarterly, and yearly notes

Planned behavior on daily pages only:

- keep native previous day, `Today`, next day, and calendar controls
- append compact buttons for `W##`, compact month label, and `YYYY`

The starting `plugin.json` mirrors the current live Daily Note collection configuration.
